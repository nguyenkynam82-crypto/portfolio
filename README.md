## ⚡ Architectural Marvels

This repository is not just a landing page; it's a testament to extreme performance optimization and modern front-end engineering.

- **Cinematic Asymmetric Layout**: Moving away from generic web clichés. Combining corporate sophistication with deep-tech brutalism.
- **Micro-interactions & Physics**: Powered by `Framer Motion` and `Lenis`, delivering hyper-smooth inertial scrolling and reactive elements without layout thrashing.
- **Aggressive Performance**: Sub-second Largest Contentful Paint (LCP) via CSS `content-visibility` and React lazy rendering. Main thread blockage is practically zero.
- **Glassmorphism & Depth**: Multi-layered ambient lighting, blurred glass layers, and dark-mode exclusive visual weight balancing.

## 💻 Quick Start

### Prerequisites
- `Node.js` >= 18.x
- `npm` or `yarn` or `pnpm`

### Installation

```bash
# Clone the repository
git clone https://github.com/DonQuaan/donquaan.git

# Navigate into the project directory
cd donquaan

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Build for Production

```bash
# This will generate the highly-optimized static files in the /dist folder
npm run build
```
